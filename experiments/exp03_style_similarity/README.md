# 시험 3: style_similarity (Match Score 35점 구간)

## 목적

Match Score 100점 중 최대 피쳐(35점)인 style_similarity가
완전히 LLM 주관으로 산출됨. 분포, 일관성, 방향성을 검증한다.

## 입력 → 출력

- 입력: 유저 스타일 태그 + 옷 이미지 묘사 텍스트
- 출력: 0~35점 + reason 한 줄

## 시험 항목

### a) 점수 분포 확인 (가장 중요)
- 스타일 태그 5종 × 옷 묘사 20종 = 100개 조합
- 같은 조합 3회 반복 → 점수 변동 폭(분산) 측정
- 기대: 0~35 전 구간에 고르게 분포. 15~25에 몰리면 변별력 없음
- 히스토그램 시각화

### b) 일관성 검증
- temperature=0 고정, 3회 반복 시 표준편차
- ≤3점 OK, ≥5점이면 프롬프트 개선 필요

### c) 방향성 검증 (sanity check)
- 자명한 케이스 20개:
  - 캐주얼 선호 + 캐주얼 옷 → ≥25점
  - 페미닌 선호 + 스트릿 옷 → ≤15점

### d) 가시성 제약 준수 확인
- visibility="부분 가림" → ≤20점
- visibility="불량" → ≤8점
- 프롬프트에 명시해도 LLM이 무시하는 경우 있음 → 별도 체크
- cap 적정성도 검토

### e) reason 문장 품질
- context_sentences의 style_context로 직접 사용됨
- 자연스러운 한국어인지, 길이 적절한지, system_prompt 톤과 충돌 없는지
