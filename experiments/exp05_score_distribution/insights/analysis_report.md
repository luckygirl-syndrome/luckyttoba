## 2. 피쳐별 기여도 — 핵심 발견

| 피쳐 | 평균 기여도 (raw) | 비율 |
|------|-------------------|------|
| 할인율 (discount) | 0.2424 | **50.8%** |
| 평점 (rating) | 0.1819 | **38.1%** |
| 리뷰수 (review) | 0.0221 | 4.6% |
| 좋아요수 (like) | 0.0227 | 4.8% |
| 마케팅 (marketing) | 0.0082 | **1.7%** |

### 인사이트

**할인율 + 평점이 전체의 89%를 지배한다.**

- 공식 가중치는 할인율 35%, 평점 20%이지만, 실제 기여도는 51%, 38%로 비율이 더 극단적
- 원인: 리뷰수/좋아요수는 로그 스케일 + 1.5 거듭제곱이라 대부분의 상품에서 0.1~0.3 수준으로 압축됨. 반면 할인율/평점은 선형에 가까워 값 범위가 넓음
- **마케팅 기여도 1.7%는 과소추정**: 키워드 매칭 히트율이 11.3%에 불과 (trend_hype 1.8%, bundle 4.8%, confidence 5.8%). "완판임박", "누적 10만장" 같은 변형 표현을 못 잡음
  - → 시험 1 마케팅 트리거 평가 완료 후 LLM 분류 결과로 재계산하면 마케팅 기여도가 올라갈 것으로 예상

### 조치 검토 필요 사항

- 리뷰수/좋아요수의 로그+거듭제곱 정규화가 너무 공격적이어서 변별력이 약함
- 단, 이는 "충동 자극" 관점에서 리뷰 100개나 5000개나 비슷하게 "많다"고 느끼는 심리를 반영한 설계이므로, 의도된 것일 수 있음
- → 팀 논의 후 결정

| 4 | DUTO | 42 | 41.3 |
| ... | | | |
| 15 | DIMO | 27 | 26.7 |
| 16 | NIMO | 27 | 26.7 |

→ `impulse_16types_boxplot.png` 참조

### 인사이트

**D/N 축(도파민 vs 필요)은 거의 차이가 없다.**
- D형 평균 34.5 vs N형 평균 34.4
- Impulse Score 공식에서 D/N의 역할이 title_marketing_m (1.1 vs 0.9)뿐인데, 마케팅 기여도 자체가 1.7%라 영향이 미미

**가장 큰 차이를 만드는 축: U/I와 T/M**
- U형(타인 검증)은 review_count_m, like_count_m에서 1.2~1.3 배율을 받고, T형(동조)은 추가 1.2배
- M형(차별화)은 리뷰/좋아요에서 음수 점수를 받아 전체적으로 낮아짐
- **UTE 조합이 가장 높고, IMO 조합이 가장 낮은 패턴이 일관적**

**M형 0점 몰림은 없다.**
- M형 평균: 29.4 (비M형: 39.6). 차이는 있지만 0점 몰림은 아님
- M형의 review_contrib 평균이 -0.0198로 실제로 음수이긴 하나, 할인율/평점 기여가 여전히 지배적이라 전체 점수는 양수 유지

## 4. 플랫폼별 분석

| 플랫폼 | 평균 | 중앙값 | 표준편차 |
|--------|------|--------|----------|
| 지그재그 | 38.9 | 39 | 11.6 |
| 무신사 | 33.9 | 33 | 15.4 |
| 에이블리 | 30.6 | 30 | 15.2 |

→ `impulse_platform.png` 참조

### 인사이트

- **지그재그가 가장 높음**: 좋아요 기능이 없어서 like_contrib=0이 오히려 이점. M형에서 like 감점이 없고, 비M형에서도 like 기여가 원래 낮아서 영향 적음. 대신 지그재그 상품의 할인율/평점이 상대적으로 높은 것이 주 원인
- **에이블리가 가장 낮음**: rating 분포가 낮고 (많은 상품이 4.0 부근), review_count도 상대적으로 높아 M형 감점이 큼
- **지그재그 std가 낮음(11.6)**: like_count 변동이 없어서 분산 요인이 하나 빠짐

## 5. 구간 설계

### Impulse Score 5구간

| Tier | 범위 | 이름 | 비율 |
|------|------|------|------|
| 1 | 0~21 | 매우 낮음 | 20.7% |
| 2 | 22~30 | 낮음 | 20.4% |
| 3 | 31~39 | 보통 | 21.9% |
| 4 | 40~47 | 높음 | 17.7% |
| 5 | 48~100 | 매우 높음 | 19.3% |

- 백분위(P20/P40/P60/P80) 기반으로 경계를 정수 구간에 맞춤 (반올림 없이 실제 백분위 사용)
- 최대 편차 2.3%p로 5구간이 ~20%씩 균등 분포
- 시험 1(마케팅 트리거)/3 반영 후 분포가 변하면 구간도 재확정 예정

### Match Score (3피쳐 소계, 참고)

- 범위: 13~65 (65점 만점)
- 평균: 34.0, 중앙값: 34.0
- style_similarity(0~35) 미포함 → 시험 3 후 합산하여 최종 100점 만점 구간 확정

- D형/N형 간 산점도 상의 위치 차이 없음 (위에서 분석한 것과 일관)
