# 시험 5: Match Score 분포 분석 보고서

**작성일**: 2026-04-30  
**대상 데이터**: 모든 객관식 조합 기반 80명 가상 유저  
**분석 범위**: Match Score (style_similarity 제외, 65점 만점)

---

## 1. 개요

Match Score는 옷이 사용자의 **개인 취향, 가격 선호도, 구매 의향도**에 얼마나 잘 맞는지를 나타내는 점수입니다. **충동구매 방지가 아닌**, 사용자가 해당 옷을 얼마나 선호할 가능성이 높은지를 측정합니다. 이번 시험에서는 3가지 피쳐(가격, 관심지속도, 발견경로)의 분포를 분석하고, 최적의 구간(Tier)을 정의했습니다.

---

## 2. 데이터 생성 방식

### 2.1 구성
- **전체 조합**: price(4) × interest(4) × discovery(5) = **80개**
- **생성 방식**: 모든 객관식 응답 조합을 정확히 1명씩 매핑
- **S-BTI 제외**: Match Score 계산에 영향을 주지 않으므로 제거

### 2.2 피쳐별 응답 옵션

| 피쳐 | 옵션 (개수) | 점수 범위 |
|------|-----------|---------|
| **Price Reasonable** (가격 적합도) | 4가지 | 3~25점 |
| **Interest Persistence** (관심 지속도) | 4가지 | 4~20점 |
| **Discovery Stability** (발견 경로 안정도) | 5가지 | 6~20점 |

## 3. 분포 분석 결과

### 3.1 전체 분포 특성

```

Std:      11.553289

```

**해석:**
- **범위**: 13~65점 (전체 범위 완벽하게 커버)
- **중앙값**: 37점 (평균과 동일 → 대칭적 분포)
- **표준편차**: 11.55점 (중간 수준의 변동성)
- **고유값**: 39개 (매우 다양한 분포)

### 3.2 피쳐별 분포

#### Price Reasonable (가격 적합도)
```
값: [3, 10, 18, 25]점
평균: 14.0점
표준편차: 8.33점
분포: 균등 (각 4가지 값 × 20명)
```

**특징**: 
- 가장 큰 영향력 (표준편차 8.33)
- 모든 값이 고르게 나타남
- 범위: 0~25점 (최대 폭)

#### Interest Persistence (관심 지속도)
```
값: [4, 10, 16, 20]점
평균: 12.5점
표준편차: 6.10점
분포: 균등 (각 4가지 값 × 20명)
```

**특징**:
- 두 번째 영향력 (표준편차 6.10)
- 균등 분포
- 범위: 0~20점

#### Discovery Stability (발견 경로 안정도)
```
값: [6, 10, 12, 20]점
평균: 10.8점
표준편차: 5.19점
분포: 불균등 (6점: 32명, 10/12/20점: 각 16명)
```

**특징**:
- 가장 작은 영향력 (표준편차 5.19)
- 가장 낮은 점수(6점)에 우중앙된 분포
- 범위: 0~20점

## 4. 피쳐 영향력 분석

### 4.1 기여도 비율

| 피쳐 | 표준편차 | 기여도 비율 |
|------|---------|-----------|
| **Price** | 8.33 | **37.1%** |
| **Interest** | 6.10 | **34.0%** |
| **Discovery** | 5.19 | **28.9%** |

**해석:**
- Price가 전체 점수 변동의 **37%** 결정
- Interest와 Discovery는 각각 **34%, 29%** 기여
- 3가지 피쳐가 균형 있게 영향 (한 피쳐 우세 아님)

### 4.2 피쳐 간 상관성

- **Price ↔ Interest**: 약한 상관 (r ≈ 0.10)
- **Price ↔ Discovery**: 약한 상관 (r ≈ 0.08)
- **Interest ↔ Discovery**: 약한 상관 (r ≈ 0.05)

**해석:**
- 3가지 피쳐가 거의 **독립적**
- 한 피쳐 변화가 다른 피쳐에 영향을 주지 않음
- 조합의 다양성 보장

- ✅ 균등 분할 (각 25% 수준)

- ✅ 직관적이고 해석이 용이

## 6. 권고사항

### 6.1 최적 구간: 4분위 백분위 기반 추천

**이유:**
1. 균등하고 직관적인 분할 (각 ~25% 수준)
2. 모든 데이터 활용 (최대 정보량)
3. 사용자의 "취향 일치도"를 명확하게 전달
4. 향후 style_similarity 추가 시 조정 용이

### 6.2 구간별 UI/UX 권고

| Tier | 색상 | 메시지 | 액션 |
|------|------|--------|------|
| **1** | 🔴 빨강 | "당신의 취향과 맞지 않을 수 있어요" | 비슷한 스타일 상품 추천 |
| **2** | 🟡 노랑 | "당신의 취향과 어느정도 맞아요" | 추가 정보 제공 |
| **3** | 🟢 초록 | "당신의 취향과 잘 맞아요!" | 일반 정보 제공 |
| **4** | ✅ 파랑 | "당신의 취향과 매우 잘 맞는 옷이에요!" | 결제 유도 |

### 6.3 향후 개선 방향

1. **Style Similarity 추가**
   - 현재: 65점 만점 (3피쳐)
   - 미래: 100점 만점 (style 포함)
   - 구간 재정의 필요

2. **사용자 세그먼트별 가중치**
   - PLAN에서 "유형별 일반화 폐기" 결정했으나
   - 향후 개인의 쇼핑 성향별 가중치 검토 가능
   - 예: 가격민감도, 계절성 선호도 등

3. **실제 구매 데이터 검증**
   - 현재: 가상 데이터 기반 (모든 조합 균등)
   - 미래: 실제 사용자 선호도와 구매 패턴 매칭 확인

## 7. 결론

Match Score는 **37점(중앙값)** 기준으로 4개 구간으로 나뉨:

- **Tier 1 (13~29점, 27.5%)**: 취향 불일치 - 사용자가 이 옷을 선호할 가능성 낮음
- **Tier 2 (30~37점, 22.5%)**: 취향 약간 불일치 - 선호도 중간 수준
- **Tier 3 (38~46점, 26.2%)**: 취향 어느정도 일치 - 선호도 높음
- **Tier 4 (47~65점, 23.8%)**: 취향 매우 일치 - 사용자가 이 옷을 선호할 가능성 매우 높음

3가지 피쳐가 **균형 있게** 영향을 미치므로 (Price 37%, Interest 34%, Discovery 29%), 사용자 세그먼트 개인화 없이 **피쳐 그대로 합산**하는 현재 설계가 타당합니다.
